ssss

  • This tutorial project is based on the Website/Web Service template in Node.js by Oren Golan.
  • This is a simple HTTP server, which serves it's content on port 8080. Main file is server.js. It registers routers for particular URL patterns.
  • Static HTML content is served from public_html folder through router routes/static.js.
  • Index file is created from an EJS template templates/index.ejs and served by router routes/tutorial.js. Data for the template is taken from tutorial/* files, each containing code walkthrough of one section.
